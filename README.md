# University Campus Network Design

## Overview

This project presents a University Campus Network designed and simulated using Cisco Packet Tracer. The network demonstrates how a large educational institution can be connected through a structured, scalable, and secure network architecture.

## Features

- Campus-wide network topology
- Department-wise network segmentation
- VLAN configuration
- Routing between departments
- Switching infrastructure
- End-device connectivity
- Scalable network design
- Cisco Packet Tracer simulation

## Technologies Used

- Cisco Packet Tracer
- Cisco Routers
- Cisco Switches
- VLANs
- Routing Protocols
- Ethernet Networking

## Network Components

The network includes:

- Administrative Department
- Academic Departments
- Computer Labs
- Faculty Offices
- Student Network
- Core Routers
- Layer 2/Layer 3 Switches
- End Devices (PCs, Servers, Printers)
